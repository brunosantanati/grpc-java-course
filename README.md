## Evans Commands  

```
evans
evans --host localhost --port 50052 --reflection repl
show package
show service
show message

To call an endpoint you can use:
call endpoint-name

examples:
call sum
call primes
call avg (ctrl + d to stop entering values)
call max (ctrl + d to stop entering values)
```
